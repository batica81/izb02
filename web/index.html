Frontend application

Short description

Create a responsive web frontend application for managing the articles and comments. Registered user can post Artiles (artiles are up to 255 characters) and post comments on their and someone else's articles (comments are up to 255 characters).

Specification

General requirements

  1. Responsive UI, based on Bootstrap 3
  2. App should support Desktop and mobile at least
  3. App should consume specified API
  4. Please make frontend user registration form with frontend form validation.

User stories

  1. Visitors (everyone) can list all articles and comments on them. They can not post newarticles or comment.
  2. Visitors can register can register and become Users (i.e. Posters).
  3. Posters can make new articles and comment on articles.

BONUS user stories

  1. Posters can change or delete their Articles or Comments.
  2. Posters can change their profile information.
  3. Any improvement you decide it is beneficial for UX.
